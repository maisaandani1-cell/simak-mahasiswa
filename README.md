# 📚 Sistem Akademik Mahasiswa (SIMAK)

## 📌 Deskripsi Proyek
SIMAK (Sistem Akademik Mahasiswa) adalah aplikasi berbasis web yang dirancang untuk membantu mahasiswa dalam mengelola kegiatan akademik secara digital. Sistem ini menyediakan berbagai fitur penting seperti pengisian KRS, evaluasi dosen, konsultasi akademik, pengajuan revisi nilai, dan informasi tagihan kuliah.

---

## 🎯 Tujuan
- Mempermudah proses pengisian KRS
- Menyediakan sistem evaluasi dosen (EDOM)
- Memfasilitasi konsultasi akademik
- Mengelola pengajuan revisi nilai
- Menampilkan informasi tagihan kuliah

---

## ⚙️ Fitur Utama
- 📥 Pengisian KRS (Kartu Rencana Studi)
- 📝 Evaluasi Dosen oleh Mahasiswa (EDOM)
- 💬 Konsultasi Bimbingan Akademik
- 🔄 Pengajuan Revisi Nilai
- 💳 Informasi dan Bukti Tagihan Kuliah
